Author:

Conforms to:

Superclass:
NSObject

Introduction

A very easy way to have access to all the features of the NabuOpenSDK

Discussion

The NabuManager is a singleton whose shared instance makes the access to the NabuOpenSDK very easy and elegant

authenticateWithAppId:andURISchemeCallback:

Method for authenticating with the Razer Nabu 

- (BOOL)authenticateWithAppId:(NSString *)appId 
        andURISchemeCallback:(NSString *)uriScheme;
Parameters
appId

The app id which has been registered on developer.razerzone.com/nabu

uriScheme

The app's URL type for communicating with other apps

Return Value

True if successfully authenticated

Discussion

Works with both the Nabu and the Nabu X

WARNING:

Should be sent just once, at the first run of the app

checkIfAuthenticated

Method for checking if the app is authenticated 

- (BOOL)checkIfAuthenticated;
Return Value

True if successfully authenticated
